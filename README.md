# 🀄 StudyChinese

> WPF-приложение для изучения китайского языка: интервальное повторение,
> AI-чат с настройкой уровня, и викторина в формате «Своей игры».

---

## ✨ Функциональность

### 🃏 Карточки — Интервальное повторение (Spaced Repetition)

Алгоритм по образу Anki: система сама определяет когда показать карточку повторно.

**Алгоритм выбора карточки:**
1. Сначала показываются карточки у которых **истекло время повтора**
2. Затем карточки с **наименьшим статусом** знания (случайно из группы)

**Статусы и интервалы:**

| Оценка | Следующий повтор | Статус |
|---|---|---|
| Не помню | через 2 минуты | 🔴 Учу |
| С трудом | через 7 минут | 🟡 Почти |
| Легко | через 1 день | 🟢 Знаю |

**Управление наборами:**
- Создание своих наборов карточек (иероглиф ↔ перевод)
- Импорт из текстового файла в формате `иероглиф/перевод` построчно
- Встроенные наборы HSK1, HSK2, HSK3
- Статистика прогресса: не начато / учу / выучено

---

### 🎮 Квиз — «Своя игра» (Jeopardy-формат)

Викторина в формате телеигры «Своя игра» с темами и стоимостью вопросов.

**Как устроено:**
- Таблица с темами по строкам и вопросами по столбцам
- Стоимость вопроса растёт по столбцам (множитель × 10)
- Нажимаешь на клетку → появляется вопрос → открываешь ответ → клетка исчезает
- Квизы хранятся как **JSON-файлы**, загружаются динамически

**Редактор квизов:**
- Создание квиза через UI в 2 шага (настройки → вопросы)
- Добавление тем и вопросов с ответами через карточки
- Перестановка вопросов стрелками ↑↓
- Сохранение в JSON с сериализацией через Newtonsoft.Json

---

### 💬 AI-чат (DeepSeek)

Практика разговорного китайского с AI в двух режимах.

**Режимы:**
- **Teacher** — строгий преподаватель, объясняет грамматику на китайском
- **Friend** — китайский друг, общается неформально

**Уровни сложности (выбираются в чате):**

| Уровень | Словарный запас | Длина фраз |
|---|---|---|
| Beginner | HSK1–HSK2 | 5–8 иероглифов |
| Intermediate | HSK3–HSK4 | 10–15 иероглифов |
| Advanced | HSK5–HSK6 | 15–30+ иероглифов, 成语 |

- AI **всегда отвечает только на китайском** — полное погружение
- Сообщения пользователя отображаются справа (голубые), AI — слева (розовые)
- Отправка по кнопке или клавишей **Enter**

## 🛠️ Технологии

| Библиотека | Назначение |
|---|---|
| **WPF / .NET 9** | Desktop UI, динамическое построение интерфейса в коде |
